Error Page Help

This page appears when an error occurs during file processing or analysis.

What to do:

  1. Check your file format - make sure it's a valid CSV file
  2. Verify the file contains the required columns for analysis
  3. Ensure your file isn't corrupted or too large
  4. Try returning to the home page and uploading again

If problems persist, please check the console for more detailed error information.
